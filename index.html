Programming 101

By: Dominic Varchetto

About section:

This webpage will talk about the basics of coding from someone with no experience whatsoever. There will be different sections talking about HTML, CSS, JavaScript, and how to tutorials with animations, images, and much more. Not only will there be information regarding the different steps in coding, there will be some examples of projects and coding that I have done for class assignments. Overall, this webpage is to help others who are inexperienced get a sense of what coding is like, what goes into coding, how to code, and some final products that implemented code.

Technology Overview:

This webpage will use Github, Git, JavaScript, HTML, CSS, W3Schools, and other resources to help create this.
